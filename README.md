# StoryOP
A Behaviour-Driven Development framework for Delphi/DunitX. This project is based on StoryQ, with a fluent syntax, expressive human-readable test output, an easy single file wrapper for DUnitX that works with TestInsight or your preferred test runner, and designed to capture behaviour/features/user-stories directly in source code if needed.
http://storyq.codeplex.com/
